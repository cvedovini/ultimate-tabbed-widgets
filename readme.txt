== Description ==

A plugin that allows you to create widget areas that can be turned into tabs or
accordion using widgets or shortcodes.

Use the settings page to define as many widgets areas as you want then use the
"Ultimate Tabbed Widgets" widget to show those areas as a set of tabs or an
accordion in your sidebars.

You can also use the `[utw widgets="name of area" template="tabs|accordion"]`
shortcode to show a set of tabs or an accordion inside a post or a page.


== Installation ==

This plugin follows the [standard WordPress installation
method](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins):

1. Upload the `ultimate-tabbed-widgets` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
